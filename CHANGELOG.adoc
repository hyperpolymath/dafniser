== [0.1.0] - 2026-03-21

=== Phase 1 Complete

- Spec parser for requires/ensures/invariant/decreases/ghost clauses
- Dafny source generation with verification annotations
- Z3 verification compiler integration
- Multi-target compilation stubs (C#, Java, Go, Python, JavaScript)
- Bespoke manifest parsing for function specifications
- Integration tests for spec parsing and Dafny codegen

=== Added (scaffold — 2026-03-20)

- Initial project scaffold from rsr-template-repo
- CLI with subcommands (init, validate, generate, build, run, info)
- Manifest parser (`dafniser.toml`)
- Codegen engine stubs
- ABI module (Idris2 proof type definitions)
- Library API for programmatic use
- Full RSR template (17 CI workflows, governance docs, bot directives)
- README.adoc with architecture overview
